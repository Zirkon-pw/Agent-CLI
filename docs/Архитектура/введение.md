# Архитектура Agent CLI

Эта папка описывает, как Agent CLI должен быть реализован как консольное приложение на Go.

В отличие от раздела `Устройство`, здесь зафиксирована инженерная модель:

- из каких слоев состоит CLI;
- как разложить код по пакетам Go;
- как выполняются команды;
- как устроены файловое хранилище и runtime;
- где живут статусы задач, шаблоны промптов и `.yml`-уточнения.
